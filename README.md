# Last Outpost

3D First-Person Defense Game built with Unity.

## 🎯 Project Overview

This project is a final synthesis assignment for a Game Development course.

The objective is to defend a central core against waves of enemies using first-person weapons.

The player wins by surviving the last wave.
The player loses if:
- Player health reaches 0
- Enemies reach and destroy the central core

## 📜 Requirements Covered

- 3 Enemy types (Inheritance from Enemy)
- 3 Gun types (Inheritance from Gun)
- Enemy targeting manager (max 3 chase player)
- Wave system with scaling difficulty
- Event-driven UI (no Update polling for resources)
- FPS movement system
- Post-processing
- VFX
- Audio system
- Tooltips
- Clean architecture
- No console errors

## 🧠 Architecture Philosophy

The project is built using:
- OOP principles
- Inheritance
- Polymorphism
- Manager-based architecture
- Event-driven UI updates
